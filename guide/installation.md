# Installation

Avant de commencer l'installation de votre première machine virtuelle, je vous invite fortement à vous renseigner sur ce qu'est une machine virtuelle. 
Vous trouverez dans ce guide la façon dont j'ai procédé pour créer ma propre machine et les erreurs que j'ai pu comettre ou rencontré. J'espère que cela vous sera utile.

## Récupération de debian

La première étape pour créer votre machine virutelle sera de trouver le bon debian pour l'installer.

> [!NOTE]  
> Pour trouver la bonne configuration à télécharger, il faudra voir avec ce dont vous disposer comme ordinateur. Certaines version (notamment les x64) ne fonctionnent pas sur tous les ordinateurs.
