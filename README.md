# Born2beroot

> Projet qui a pour objectif de faire découvrir le monde merveilleux de la virtualisation.

You will create your first machine in VirtualBox using specific instructions. Then, at the end of this project, you will be able to set up your own operating system while implementing strict rules.

## Versions

- [Debian](https://www.debian.org/) : 13.1.0
- [VirtualBox](https://www.virtualbox.org/) : 7.0.20
